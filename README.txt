Welcome to FreeEMS-Loader!

##########################  LINUX USERS #################################
To build and run this application please run the following commands:

1. "apt-get update && apt-get install libboost-dev libboost-system-dev"

2. Simply type "qmake && make all" and hit enter!

3. To execute your freshly built loader type "./FreeEMS_Loader"

It's as easy as that!

#########################  WINDOWS USERS ################################
1. Install MinGW

2. Install boostc++ libs(1.4.x)

3. Install QT libs(4.x.x)

4. Simply type "qmake && make all" and hit enter!
